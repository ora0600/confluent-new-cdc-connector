# Copy Data from oracle to a Kafka topic
First install first kafkaka. We need it later to load CSV data into kafka (Confluent Cloud)

```bash
brew install kcat
```

Now, we would like to export Data from Oracle with a simple query:

```bash
sqlplus ordermgmt/kafka@XEPDB1
SQL> SET LINESIZE 1000 TRIMSPOOL ON PAGESIZE 0 FEEDBACK OFF
SQL> SPOOL /tmp/orders.csv
SQL> SELECT order_id || ',' ||
       customer_id || ',"'||
       status ||'",' ||
       salesman_id || ',"'||
       TO_CHAR(order_date,'DD-MON-YYYY')||'"' AS order_date 
FROM   orders
ORDER BY order_id;
SQL> SPOOL OFF
SQL> SET PAGESIZE 14
SQL> !cat /tmp/orders.csv
```
